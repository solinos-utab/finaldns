Ringkasan Sistem

FinalDNS adalah sistem DNS Server berkinerja tinggi yang dirancang khusus untuk lingkungan ISP dan Corporate. Sistem ini menggabungkan keamanan (Security Shield), kecepatan (CDN Optimization), dan keandalan (High Availability).

1. Dashboard & Monitoring

Dashboard utama memberikan pandangan real-time mengenai kesehatan jaringan DNS Anda:

  • Traffic Analysis: Grafik live yang menampilkan Query per Second (QPS), Total Queries, dan Ratio Blokir.
  • System Status: Monitoring penggunaan CPU, RAM, dan Disk secara real-time.
  • Top Clients & Domains: Analisis klien yang paling aktif dan domain yang paling sering diakses.

2. Security Shield (Sistem Keamanan)

Security Fitur perlindungan aktif yang dapat diaktifkan/dinonaktifkan secara instan melalui dashboard.

Komponen Security Shield:

  • Malware Protection: Memblokir domain yang terindikasi menyebarkan malware, ransomware, dan phising.
  • Internet Positif: Kepatuhan terhadap regulasi pemblokiran konten negatif (Judi, Pornografi, SARA).
  • External Threats: Perlindungan terhadap botnet, C2 servers, dan crypto miners.
  • Custom Blacklist: Daftar blokir manual yang dapat Anda atur sendiri.
Cara Menggunakan Toggle Security Shield
  1. Buka Dashboard Web GUI.
  2. Cari panel "SECURITY SHIELD" di pojok kanan atas.
  3. Klik tombol toggle untuk mengaktifkan (Hijau/Active) atau menonaktifkan (Merah/Disabled).
  4. Perubahan berlaku instan tanpa perlu restart manual.

3. Gaming Optimization (Game Mode)

Performance New

Sistem ini dilengkapi dengan Aggressive Gaming Whitelist yang memastikan trafik game online tidak akan pernah terblokir dan mendapatkan prioritas rute.

Fitur Game Mode:

  • Bypass Filtering: Domain game (Steam, Epic, Riot, EA, Mobile Legends, PUBG, dll) di-whitelist secara permanen.
  • Direct Routing: Menggunakan rute tercepat untuk koneksi ke server game, mengurangi latensi (ping).
  • EA / FIFA Fix: Perbaikan khusus untuk masalah koneksi "prepaired game" pada EA FC25/FIFA.

Config File: /etc/dnsmasq.d/05-gaming-whitelist.conf

4. Streaming & Smart TV Optimization

Performance New

Optimasi khusus untuk perangkat Android TV, Google TV, dan layanan streaming populer.

Perbaikan Utama:

  • MTU Fix (Packet Fragmentation): Mengatur edns-packet-max=1232 untuk mencegah buffering pada jaringan PPPoE/IndiHome.
  • Connectivity Checks: Whitelist untuk pengecekan koneksi Android/Google TV (mencegah status "No Internet").
  • Supported Services: Optimasi khusus untuk Vidio, WeTV, Netflix, Disney+, Viu, Vision+.
  • NTP Sync: Memastikan sinkronisasi waktu akurat untuk DRM streaming.

Config File: /etc/dnsmasq.d/98-streaming-optimization.conf

5. CDN Optimization (Turbo Mode)

Speed New

Untuk mempercepat loading konten berat (Video, Gambar), sistem melakukan Direct Routing ke Public DNS tercepat (8.8.8.8 / 1.1.1.1) untuk penyedia konten utama.

Provider Metode Optimasi
Google / YouTube Bypass Local -> Direct 8.8.8.8
Meta (FB/IG/WA) Bypass Local -> Direct 8.8.8.8
Cloudflare / TikTok Bypass Local -> Direct 1.1.1.1 / 8.8.8.8
Netflix / Apple / Microsoft Bypass Local -> Direct 8.8.8.8

Config File: /etc/dnsmasq.d/99-cdn-optimization.conf

6. Manajemen & Troubleshooting

Restart Service

Jika Anda melakukan perubahan konfigurasi manual, Anda dapat merestart layanan DNS melalui menu "System Control" -> "Restart DNS Service" di dashboard.

Flush Cache

Jika domain yang seharusnya bisa diakses masih terblokir (atau sebaliknya), gunakan fitur "Flush DNS Cache" untuk membersihkan memori DNS.

Backup & Restore

Sistem dilengkapi dengan script auto-installer dan backup penuh. Folder backup tersimpan di /home/dns/finaldns_backup/.
